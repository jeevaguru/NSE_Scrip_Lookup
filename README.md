# NSE_Scrip_Lookup
A simple python script to lookup the last traded price, day high and day low of given listed company

## Instructions:

* Install Python (recommended: Python 3.9+), and required packages:
* `pip install nsetools requests`
* Run the script as `python check_price.py`

_This works on Windows, macOS, Linux as long as Python and dependent packages are installed._
